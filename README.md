# Cybersecurity Homelab


![image alt](https://github.com/rudrabharad/rudrabharad/blob/5fb99cdccdc6b5b5f207b74f9a3c22013919ed15/Screenshot%202025-12-27%20125956.png)

This repository documents my cybersecurity homelab built for learning and practice.
The goal is to understand networking fundamentals, TCP/IP, and basic offensive and defensive security concepts using virtual machines.

## Lab Environment
- Host OS: Windows / Linux
- Hypervisors: VirtualBox, VMware Workstation
- Guest OS:
  - Kali Linux
  - Tails Linux

## What I Am Learning
- Networking basics and TCP/IP
- Linux fundamentals
- Network scanning and reconnaissance
- Packet analysis
- Basic system hardening

## Tools Used
- Nmap
- Wireshark
- Metasploit
- Netcat
- Burp Suite

## Lab Structure
- `setup/` VM and OS installation guides
- `networking/` networking theory and notes
- `security/` offensive and defensive practices
- `labs/` hands on exercises
- `docs/` documentation and learning notes

## Disclaimer
This homelab is for educational purposes only.
All testing is done in a controlled environment that I own or have permission to use.
